# EKD
This repo contains the website for Enkel Digital.


## Project setup
Installation:
```
npm install
```
Compiles and hot-reloads for development:
```
npm run serve
```
Compiles and minifies for production:
```
npm run build
```
Run your tests:
```
npm run test
```
Lints and fixes files
```
npm run lint
```
Single command build and deploy:
```
npm run deploy
```
